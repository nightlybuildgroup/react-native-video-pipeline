# Maestro E2E flows

This directory is reserved for [Maestro](https://maestro.mobile.dev/) end-to-end flows that drive the native example apps (`apps/bare-example`, `apps/expo-example`).

Flows land with the upcoming golden-test task. Until then, this directory is intentionally empty aside from this README.

## Out of scope here

- **Playwright** is _not_ configured in this repo. Playwright is deferred entirely to the docs-site work in v0.5; do not add a Playwright dependency as part of the Maestro setup.
